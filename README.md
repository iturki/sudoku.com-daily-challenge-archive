# Sudoku.com Daily Challenge Archive

This repository contains a comprehensive archive of [Daily Challenge puzzles scraped from **Sudoku.com**](https://sudoku.com/challenges), starting from January 1st, 2020. The puzzles are stored in a custom **.sdk** text format, organized hierarchically by year and month.

## üìÇ Repository Structure

The archive is organized into folders by Year and Month to keep the repository navigable.

**Path Pattern:**
`dc-puzzles-{YYYY}/dc-puzzles-{YYYY}-{MM}/sudoku.com-dc-puzzle-{YYYY}-{MM}-{DD}.sdk`

**Examples:**
* `dc-puzzles-2020/dc-puzzles-2020-01/sudoku.com-dc-puzzle-2020-01-01.sdk`
* `dc-puzzles-2023/dc-puzzles-2023-12/sudoku.com-dc-puzzle-2023-12-25.sdk`

## üìÑ SDK File Format Specification

Each [`.sdk` file](https://sudocue.net/fileformats.php) is a plain text file containing metadata headers followed by the puzzle grid.
